# Deal Debrief Agent

## Description
Post-close analysis for won or lost deals — what worked, what didn't, decisive stakeholders, competitive dynamics, and lessons learned for institutional knowledge.

## Audience
AEs, Sales managers, Sales leaders

## Input
Account name (selects a closed opportunity)

## MCP Tools Used
- find_account
- get_opportunity_status
- get_recent_opportunity_activity
- ask_sales_ai_about_opportunity
- get_account_status
- get_engaged_people
