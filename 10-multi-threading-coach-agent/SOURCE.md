# Multi-Threading Coach Agent

## Description
Identifies single-threaded relationship risk across opportunities and recommends specific people to engage based on engagement data and org structure.

## Audience
AEs, Sales managers

## Input
Account name

## MCP Tools Used
- find_account
- get_engaged_people
- get_account_status
- get_recent_account_activity
- ask_sales_ai_about_account
- get_opportunity_status
- ask_sales_ai_about_opportunity
