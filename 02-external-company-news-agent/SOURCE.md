# External Company News Agent

## Description
Gathers, analyzes, and synthesizes external market intelligence for target accounts — public company data, financial reports, earnings calls, and strategic insights.

## Audience
AEs, CSMs

## Input
Account name

## MCP Tools Used
- find_account
- account_company_news
- ask_sales_ai_about_account
- get_account_status
- get_recent_account_activity

## Additional Tools
- Web search (for private companies and supplementary research)

- news.md (agent architecture and implementation guide)
